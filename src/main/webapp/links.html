Anleitung

1. Machen Sie Excel auf und eine neue Mappe.

2. Oben im Reiter auf Daten.

3. Aus dem Web auswählen.

4. In das Feld die gewünschte URL von dieser Seite kopieren, welche nach der Eingabe des Keys angezeigt wird, und auf Ok klicken.

5. Im Reiter oben auf Transformieren und zu Tabelle auswählen.

6. "Keine" und "Als Fehler anzeigen", danach auf OK klicken.

7. Neben Colomn1 auf die Pfeile klicken.

8. Alle Spalten auswählen und "Ursprüngliche Spaltennamen als Pröfix verwenden" abwählen, auf OK klicken

9. Auf Datei im Reiter klicken und "Schliessen und Laden" auswählen.
